# GDEG Development Roadmap

## Phase 1 — Foundation (Q3 2026)
- Smart contract development (EBT, NodeRegistry, TradingEngine)
- ZK-PoE circuit design and testing
- SST hardware prototype (10kVA)
- Backend API v1 (Node.js + PostgreSQL)
- Polygon Mumbai testnet deployment

## Phase 2 — Pilot (Q4 2026 - Q2 2027)
- Nigeria pilot: 50,000 SST nodes (Lagos, Abuja, Kano, Port Harcourt)
- NERC regulatory approval
- Trading engine mainnet launch (Polygon)
- Dashboard v1 (web + mobile)
- Seed raise: $2M

## Phase 3 — Scale (Q3 2027 - Q4 2027)
- Africa expansion: Ghana, Kenya, South Africa
- GGT governance token launch
- DAO activation
- Cross-chain bridge (Polygon <-> Ethereum <-> BSC)
- Series A: $15M

## Phase 4 — Global (2028+)
- SE Asia, South America
- EU + North America (regulatory-compliant)
- 5M node target
- Series B: $50M
- EBT as global energy reserve asset
