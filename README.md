## Overview

GDEG is an enterprise-grade blockchain platform that enables decentralized, peer-to-peer energy trading across a global network of producers and consumers. Built on smart contract technology, GDEG tokenizes energy credits and enables transparent, trustless energy exchange without intermediaries.

## Core Modules

1. Smart Contracts — ERC-20 energy credit tokenization
2. Node Registry — global decentralized energy producer tracking
3. Trading Engine — peer-to-peer energy buying and selling
4. Dashboard — real-time grid monitoring and analytics
5. Wallet Integration — MetaMask and Web3 payment support

## Architecture

- Blockchain Layer: Ethereum / EVM-compatible smart contracts
- Backend: Node.js + Express API
- Frontend: React + Web3.js
- Database: IPFS (decentralized) + PostgreSQL (off-chain)
- Oracles: Chainlink for real-world energy data feeds

## Getting Started

```bash
git clone https://github.com/rabiuhamza11/gdeg
cd gdeg
npm install
npm run dev
```
